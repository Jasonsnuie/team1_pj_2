SDC Seoul Data Center
디지털 혁신을 지원하는 고성능 데이터 센터

비지니스 성공에 필요한 핵심 기능을 제공하는 SDC를 살펴 보시고, 클라우드 전문가와 상담해보세요.

Service

높은 가용성과 안정성이 검증된 데이터 센터 서비스를 제공합니다.

SDC 서울데이터센터는 데이터 센터 설계부터 구축, 운영, 유지보수까지
IT 인프라 전 영역에 대한 풍부한 경험과 노하우를 보유하고 있습니다.

검증된 전문가

전문 솔루션

효율적 인프라 운영

최근 데이터 센터는 디지털 혁신의 핵심 기반으로, 적시에 신뢰할 수 있는 고성능 컴퓨팅 역량을 제공하고 분산된 IT 네트워트에 대한 깊은 가시성을 보장하며, AI 및 머신러닝 등의 기술을 활용하여 예측 가능한 운영을 지원해야 합니다. 서울 데이터 센터는 데이터 센터 설계부터 구축, 운영, 유지 보수까지 IT 인프라 전 영역에 대한 풍부한 경험과 노하우를 보유하고 있으며, 검증된 전문가, 그리고 전문 솔루션을 바탕으로 고객의 데이터 센터 및 IT 인프라를 안전하고 효율적으로 운영합니다.

데이터 센터/구축/이전

고객의 비즈니스 성장을 안정적으로 지원할 수 있는
데이터 센터 서비스

30년간 끊임없는 혁신으로 국내 5개 데이터 센터를 Tier Ⅳ/Ⅲ 수준으로 구축 및 운영을 하고 있습니다. 또한 차세대 데이터 센터 핵심기술, 소프트 기반 데이터 센터(SDC) 기술을 전체 데이터 센터로 확산하는 혁신을 지속하고 있습니다.

상품소개

AI&MLOps Platform (AI&MLOps 플랫폼)

AI&MLOps 플랫폼은 머신 러닝 모델의 개발, 학습, 배포 과정 전체 파이프라인의 반복적인 작업을 자동화하는 머신 러닝 플랫폼입니다. 쿠버네티스 기반의 MLOps1) 환경을 제공하며, 학습 데이터와 모델, 운영 데이터의 통합적인 관리가 가능합니다.
1) MLOps: Machine Learning Development(Dev)와 Machine Learning System Operation(Ops) 통합을 목표로 하는 ML 엔지니어링 방법론

CloudML Experiments (클라우드ML 익스페리먼트)

AI 모델 학습의 실험 정보를 기록, 비교하여 챔피언 모델을 관리하기 위한 서비스이며, 클라우드ML 익스페리먼트는 클라우드ML 노트북과 클라우드ML 스튜디오를 사용해 개발한 AI 모델을 실행하여 모델 학습이 제대로 이루어지는지 실험 및 검증 결과를 관리해 주는 서비스입니다. 대시보드를 통해 모델 별 실행 내역 및 상태를 한눈에 확인할 수 있어 편리하게 모델별 실험 결과를 비교하고 관리할 수 있습니다.

Vision API (비전 API)

비전 API는 기존 데이터의 AI 영상 분석 솔루션을 기반으로 이미지 내의 여러 정보들을 인식하고 분석해 주는 API를 제공하는 서비스로, Face Recognition API를 시작으로 다양한 API들을 지속적으로 제공할 예정입니다. Face Recognition API는 얼굴 인식 기술을 활용하여 이미지 내의 얼굴 위치를 찾아 얼굴 좌표 및 속성 정보를 추출해 줍니다. 또한 이를 기반으로 두 이미지 간의 얼굴의 유사도를 측정하는 얼굴 비교도 가능합니다.

대표 서비스

Data Analytics (데이터 애널리틱스)

쉽고 간편하게 클러스터 구성을 자동화하는 서비스를 제공합니다.

웹 기반 콘솔을 통해 사용자가 직접 컨테이너 기반으로 표준 버전의 카프카를 자동으로 설치할 수 있습니다. 생성된 카프카의 구동 상태 및 이벤트 모니터링, Broker 별 리소스 사용 정보 모니터링, 인증 관리 기능 등을 웹 기반 대시보드를 통해 편리하게 관리할 수 있습니다.

쿠버네티스 상에 클러스터 구성이 간편해집니다. 표준 버전의 카프카를 자동으로 설치할 수 있으며 쿠버네티스 애플리케이션을 활용하여 쉽고 간편하게 Confluent Platform의 클러스터 구성을 자동화할 수 있습니다. 쿠버네티스 엔진의 CLI를 통해 스케일 아웃이 가능하며, 클러스터 전체로 확장하여 가용성을 향상할 수 있습니다.
